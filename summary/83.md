# 83. Remove Duplicates from Sorted List

## [Description](https://leetcode.com/problems/remove-duplicates-from-sorted-list)

## Solution

```go
func deleteDuplicates(head *ListNode) *ListNode {
    // нужно сохранить указатель на исходный head.
    // достаточно одной переменной для операций далее
    cur := head
    /*
     * условие цикла: пока один из указателей на узел (текущий или следующий)
     * не станет nil.
     * если вынести в начало функции:
     * if head == nil {
     *      return head
     * }
     * решение будет быстрее, так как в цикле нужно будет проверять только
     * cur.Next != nil
     */
    for cur != nil && cur.Next != nil {
        // проверяем значения текущего и следующего
        // если совпали меняем указатель на следующий для следующего
        if cur.Val == cur.Next.Val {
            cur.Next = cur.Next.Next
        } else {
            cur = cur.Next
        }
    }
    // возвращаемый указатель — это всегда исходный head. он никогда не меняется.
    return head
}
```

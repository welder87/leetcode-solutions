# 287. Find the Duplicate Number

## [Description](https://leetcode.com/problems/find-the-duplicate-number/)

## Solution

```go
// Ключевая идея:
// Значения в массиве это индексы в полном массиве [1, n].
// То есть два одинаковых индекса быть не может.
// Помечаем пройденный индекс, меняя его знак.
// То есть, если мы уже посещали значение по такому индексу, значит это дубль.
// Time Complexity: O(n) или o(n)
// Space Complexity: O(1) или o(1)
//
// Минус в том, что массив приходится модифицировать, а задании сказано, что это
// нельзя делать.
// По хорошему, нужно вернуть обратно знаки в массиве вторым проходом.
func findDuplicate(nums []int) int {
    // идем линейно по массиву
    for _, num := range nums {
        // получаем индекс вычитанием из абсолютного значения единицы,
        // так как массив начинается с единицы.
        idx := abs(num) - 1
        // если значение по индексу отрицательное - значит мы нашли дубль
        if nums[idx] < 0 {
            return abs(num)
        }
        // ставим метку, что мы здесь были
        nums[idx] *= -1
    }
    return -1
}

func abs(num int) int {
    if num < 0 {
        return -num
    }
    return num
}
```

```go
// Ключевая идея:
// В быстром и медленном подходе указателя точка встречи двух указателей не
// является дублирующим числом.
// После того, как они встречаются, вы должны начать новый указатель из индекса
// 0 и перемещать оба указателя по одному шагу за раз, пока они не встретятся
// снова.
// Это второе место встречи - вход цикла, который соответствует дублирующему
// значению.
func findDuplicate(nums []int) int {
    slow, fast := 0, 0
    for {
        slow = nums[slow]
        fast = nums[nums[fast]]

        if slow == fast {
            break
        }
    }
    slow2 := 0
    for {
        slow = nums[slow]
        slow2 = nums[slow2]

        if slow == slow2 {
            return slow
        }
    }
}
```

```go
// Ключевая идея:
// Создаем массив посещенных индексов размера входного массива.
// Двигаемся по входному массиву, помечая посещенные места единицей.
// Если мы уже были в по указанному индексу - это дубль.
// Time Complexity: O(n)
// Space Complexity: O(n)
func findDuplicate(nums []int) int {
    seen := make([]int, len(nums))
    // идем линейно по массиву
    for _, num := range nums {
        // проверяем были ли мы здесь
        if seen[num-1] == 1 {
            return num
        }
        // ставим метку, что мы здесь были
        seen[num-1] = 1
    }
    return -1
}
```

```go
import "sort"

// Ключевая идея:
// Сортируем массив. Ищем повторяющиеся элементы рядом друг с другом.
// Time Complexity: O(n log n)
// Space Complexity: O(n log n) или O(n) или O(1)
func findDuplicate(nums []int) int {
    sort.Ints(nums)
    for i, num := range nums[1:] {
        if nums[i] == num {
            return num
        }
    }
    return -1
}
```

# 3. Longest Substring Without Repeating Characters

## [Description](https://leetcode.com/problems/longest-substring-without-repeating-characters/)

## Solution

```go
// Ключевая идея:
// Проверяем два соседних элемента на предмет повторения символов.
// Двигаем так дальше, пока условие соблюдается.
// Time Complexity: O(n)
// Space Complexity: O(m),где m-уникальные символы в строке.
// Есть решение с массивом и хэш-таблицей.
func lengthOfLongestSubstring(s string) int {
    ans := 0
    // преобразуем в массив рун для верной адресации по индексу
    ar := []rune(s)
    // Hash Set с уникальными символами
    un := make(map[rune]struct{}, len(ar))
    // инициализируем левый и правый указатели на один элемент
    i, j := 0, 0
    // проверяем в цикле граничное условие окончания массива для правого
    // указателя
    for j < len(ar) {
        for j < len(ar) {
            // проверяем в цикле наличие элемента в Hash Set
            if _, ok := un[ar[j]]; !ok {
                // если нет - заполняем Hash Set
                un[ar[j]] = struct{}{}
            } else {
                // если есть останавливаем цикл
                break
            }
            j++
        }
        // выбираем максимальное значение
        ans = max(ans, j - i)
        // удаляем элементу по индексу i, чтобы заново начать внутренний цикл
        delete(un, ar[i])
        // сдвигаем левый указатель
        i++
    }
    return ans
}
```

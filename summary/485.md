# 485. Max Consecutive Ones

## [Description](https://leetcode.com/problems/max-consecutive-ones/)

## Solution

```go
// Ключевая идея:
// Проверяем два соседних элемента на предмет наличия единицы.
// Двигаем так дальше, пока условие соблюдается.
// Time Complexity: O(n)
// Space Complexity: O(1)
func findMaxConsecutiveOnes(nums []int) int {
    ans := 0
    // инициализируем левый и правый указатели на один элемент
    i, j := 0, 0
    // проверяем в цикле граничное условие окончания массива для правого
    // указателя
    for j < len(nums) {
        // двигаемся дальше, если элемент не единица
        if nums[i] != 1 {
            i++
            j++
            continue
        }
        // проверяем в цикле граничное условие окончания массива для правого
        // указателя, равенство элемента единице
        for j < len(nums) && nums[j] == 1{
            j++
        }
        // выбираем максимальное значение
        ans = max(ans, j - i)
        // сдвигаем левый указатель
        i = j
    }
    return ans
}
```

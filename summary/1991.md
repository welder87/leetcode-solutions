# 1991. Find the Middle Index in Array

## [Description](https://leetcode.com/problems/find-the-middle-index-in-array/)

## Solution

```go
// Ключевая идея:
// Использовать префиксный массив, но не хранить его в памяти, а считать
// "на лету".
// Time Complexity: O(n) или o(2n)
// Space Complexity: O(1) или o(1)
func pivotIndex(nums []int) int {
    // считаем сумму всех элементов массива
    // это будет сумма справа от индекса
    r := 0
    for _, num := range nums {
        r += num
    }
    // инициализируем сумму слева от элемента по индексу
    l := 0
    // стандартно движемся по массиву
    for i, num := range nums {
        // вычитаем значение по индексу к сумме справа
        r -= num
        // сравниваем, если суммы равны - возвращаем текущий индекс
        if r == l {
            return i
        }
        // прибавляем значение по индексу к сумме слева
        l += num
    }
    return -1
}
```

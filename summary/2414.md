# 2414. Length of the Longest Alphabetical Continuous Substring

## [Description](https://leetcode.com/problems/length-of-the-longest-alphabetical-continuous-substring/)

## Solution

```go
// Ключевая идея:
// Исходя из условия, что это алфавит "abcdefghijklmnopqrstuvwxyz", то разница
// между рунами алфавита - единица.
// Time Complexity: O(n)
// Space Complexity: O(n)
func longestContinuousSubstring(s string) int {
    ans := 0
    // преобразуем в массив рун для верной адресации по индексу
    ar := []rune(s)
    // инициализируем левый и правый указатели на элементы рядом друг с другом
    i, j := 0, 1
    // проверяем в цикле граничное условие окончания массива для левого
    // указателя
    for i < len(ar) {
        // проверяем в цикле граничное условие окончания массива для правого
        // указателя, разницу между соседними элементами.
        for j < len(ar) && ar[j] - ar[j-1] == 1 {
            j++
        }
        // выбираем максимум
        ans = max(ans, j - i)
        // сдвигаем границы
        i = j
        j++
    }
    return ans
}
```

# 2181. Merge Nodes in Between Zeros

## [Description](https://leetcode.com/problems/merge-nodes-in-between-zeros/)

## Solution

```go
// Ключевая идея:
// У нас есть ограничения в виде узлов с нулевыми значениями.
// В узлы с нулевыми значениями будем записывать сумму последующих узлов с
// ненулевыми значениями и отбрасывать такие узлы.
func mergeNodes(head *ListNode) *ListNode {
    // "нулевой" узел (узел с изначальным нулевым значением) и следующий за ним
    // с ненулевым (по условию задачи это всегда так)
    zero, cur := head, head.Next
    // условие цикла: внутри самого цикла, так как удобнее поймать граничное
    // условие
    for {
        // условие цикла: пока значение текущего узла ненулевое
        for cur.Val != 0 {
            // агрегируем в узел с нулевым значением сумму последующих узлов
            zero.Val = zero.Val + cur.Val
            // разрываем связь и двигаем указатель дальше (удаляем узел)
            zero.Next = cur.Next
            cur = cur.Next
        }
        // мы окончим цикл на новом "нулевом" узле, поэтому нужно ещё раз
        // сдвинуть указатель
        cur = cur.Next
        // условие выхода из цикла - когда список закончился
        if cur == nil {
            break
        }
        // двигаемся по "нулевым" узлам дальше.
        zero = zero.Next
    }
    // в итоге здесь будет последний "нулевой" узел, который нужно убрать
    zero.Next = nil
    return head
}
```

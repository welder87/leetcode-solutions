# Array

## Микропаттерны

### Sentinel value (Initial boundary value)

В начало массива виртуально добавляется элемент с минимально или максимально
возможным значением. Это упрощает последующий код.

Для python:

```py
inf = float('inf')
neg_inf = float('-inf')
```

Для go стоит учитывать ограничения по размеру типа, 32-битная система
или 64 битная.

```go
inf := math.MaxInt64 // или MaxInt32
negInf := math.MinInt64 // или MinInt32
```

Для рун и байтов можно использовать zero value

```go
var zero_rune rune
```

## Паттерны

### Max value (Min value)

Поиск максимального и минимального элемента за `O(n)`.

Универсальное решение - искать индекс.
Можно одновременно получить максимальный и минимальный элементы, а также их
индексы.

```go
func findIndexOfLargestElement(arr []int) int {
    idx := -1
    for i, num := range arr {
        if idx == -1 || num > arr[idx] {
            idx = i
        }
    }
    return idx
}
```

Только для непустого массива.

```go
func largestElement(arr []int) int {
    mx := arr[len(arr)-1]
    for i := range len(arr) - 1 {
        mx = max(arr[i], mx)
    }
    return mx
}
```

## Reverse

```go
// Полный реверс массива с помощью двух указателей
func reverse(nums []int) {
    i, j := 0, len(s) - 1
    // важное условие! не разницы четное или нечетное количество элементов.
    for l < r {
        // делаем swap, то меняем местами значения
        nums[l], nums[r] = nums[r], nums[l]
        // двигаем левый указатель вправо
        l++
        // двигаем правый указатель влево
        r--
    }
}
```

```go
// Реверс части массива с помощью двух указателей
func reverse(nums []int, l int, r int) {
    // здесь [l, r] - это математический отрезок, то есть
    // либо отрезок в информатике [l, r), тогда нужно добавить в код:
    // r--
    for l < r {
        nums[l], nums[r] = nums[r], nums[l]
        l++
        r--
    }
}
```

## Rotate

### Left Rotation

1. Развернуть первые k элементов - `reverse(arr, 0, k-1)`
2. Развернуть оставшиеся - `reverse(arr, k, n-1)`
3. Развернуть всё - `reverse(arr, 0, n-1)`

### Right Rotation

1. Развернуть всё - `reverse(arr, 0, n-1)`
2. Развернуть первые k элементов - `reverse(arr, 0, k-1)`
3. Развернуть оставшиеся - `reverse(arr, k, n-1)`

## Prefix Sum

```go
func makePrefixSum(nums []int) []int {
    // выделяем префиксный массив размером N+1, где элемент с индексом 0 всегда
    // равен нулю
    px := make([]int, len(nums) + 1)
    // проходим, заполняя префиксные суммы с первого индекса
    for i, num := range nums {
        px[i+1] = px[i] + num
    }
    return px
}
```

## Микропаттерны

### Sentinel value (Initial boundary value)

В начало массива виртуально добавляется элемент с минимально или максимально
возможным значением. Это упрощает последующий код.

Для python:

```py
inf = float('inf')
neg_inf = float('-inf')
```

Для go стоит учитывать ограничения по размеру типа, 32-битная система
или 64 битная.

```go
inf := math.MaxInt64 // или MaxInt32
negInf := math.MinInt64 // или MinInt32
```

Для рун и байтов можно использовать zero value

```go
var zero_rune rune
```

## Reverse

```go
// Полный реверс массива с помощью двух указателей
func reverse(nums []int) {
    i, j := 0, len(s) - 1
    // важное условие! не разницы четное или нечетное количество элементов.
    for l < r {
        // делаем swap, то меняем местами значения
        nums[l], nums[r] = nums[r], nums[l]
        // двигаем левый указатель вправо
        l++
        // двигаем правый указатель влево
        r--
    }
}
```

```go
// Реверс части массива с помощью двух указателей
func reverse(nums []int, l int, r int) {
    // здесь [l, r] - это математический отрезок, то есть
    // либо отрезок в информатике [l, r), тогда нужно добавить в код:
    // r--
    for l < r {
        nums[l], nums[r] = nums[r], nums[l]
        l++
        r--
    }
}
```

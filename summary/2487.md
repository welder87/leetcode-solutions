# 2487. Remove Nodes From Linked List

## [Description](https://leetcode.com/problems/remove-nodes-from-linked-list/)

## Solution

```go
// Ключевая идея:
// Реверс списка даст возможность реализовать условие задачи.
// После реверса, будем удалять узлы, у которых значения меньше у предыдущего
// узла.
// Time Complexity: O(3n)
// Space Complexity: O(1)
func removeNodes(head *ListNode) *ListNode {
    // делаем реверс на основе задачи 206
    head = reverse(head)
    // нужно оставить голову для обратного реверса, поэтому копируем указатель
    cur := head
   // условие цикла: пока указателей на следующий узел не nil.
    for cur.Next != nil {
        // сравниваем значения в текущем и следующем узлах
        // если значение в текущем больше, значит разрываем связь со следующим
        // узлом
        if cur.Val > cur.Next.Val {
            cur.Next = cur.Next.Next
        } else {
            // в остальных случаях двигаемся дальше
            cur = cur.Next
        }
    }
    // делаем реверс на основе задачи 206
    return reverse(head)
}
```

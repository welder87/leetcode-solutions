# 2351. First Letter to Appear Twice

## [Description](https://leetcode.com/problems/first-letter-to-appear-twice/)

## Solution

```go
// Ключевая идея:
// Складываем символы в set, пока не найдем первый дубль.
// Time Complexity: O(n) или o(n)
// Space Complexity: O(n) или o(m)
func repeatedCharacter(s string) byte {
    mp := make(map[rune]struct{})
    for _, char := range s {
        if _, ok := mp[char]; ok {
            return byte(char)
        }
        mp[char] = struct{}{}
    }
    return 0
}
```

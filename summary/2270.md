# 2270. Number of Ways to Split Array

## [Description](https://leetcode.com/problems/number-of-ways-to-split-array/)

## Solution

```go
// Ключевая идея:
// Использовать префиксный массив, но не хранить его в памяти, а считать
// "на лету".
// Time Complexity: O(n) или o(2n)
// Space Complexity: O(1) или o(1)
func waysToSplitArray(nums []int) int {
    // считаем сумму всех элементов массива
    // это будет сумма справа
    r := 0
    for _, num := range nums {
        r += num
    }
    ans := 0
    // инициализируем сумму слева
    l := 0
    for i, num := range nums {
        // прибавляем значение к сумме слева
        l += num
        // вычитаем значение к сумме справа
        r -= num
        // проверяем условие задачи
        if l >= r && i < len(nums) - 1 {
            // инкрементириуем ответ
            ans++
        }
    }
    return ans
}
```

# 142. Linked List Cycle II

## [Description](https://leetcode.com/problems/linked-list-cycle/)

## Solution

```go
type ListNode struct {
     Val int
     Next *ListNode
}
// Ключевая идея:
// Использовать быстрый и медленный указатели, пока быстрый указатель и
// следующий за ним не nil. Если в какой-то момент быстрый и медленный указатели
// равны - значит цикл есть. Далее двигаемся одинаково от изначальной головы и
// медленного указателя пока они не станут равны.
func detectCycle(head *ListNode) *ListNode {
    // инициализация указателей головой списка
    fast, slow := head, head
    // условие цикла: пока быстрый указатель и следующий за ним не nil
    for fast != nil && fast.Next != nil {
        // сдвигаем на 2 узла вперед
        fast = fast.Next.Next
        // сдвигаем на 1 узел вперед
        slow = slow.Next
        // проверяем равенство указателей
        if slow == fast {
            ans := head
            // условие цикла: пока два указателя не совпадут
            for ans != slow {
                ans = ans.Next
                slow = slow.Next
            }
            return ans
        }
    }
    return nil
}
```

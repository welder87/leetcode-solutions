# 189. Rotate Array

## [Description](https://leetcode.com/problems/rotate-array/)

## Solution

```go
// Ключевая идея:
// Реверс массива. Сначала полный реверс массива. Затем реверс левой части
// массива до k.Затем реверс правой части массива до k.
func rotate(nums []int, k int)  {
    if k > len(nums) {
        k %= len(nums)
    }
    reverse(nums, 0, len(nums) - 1)
    reverse(nums, 0, k - 1)
    reverse(nums, k, len(nums) - 1)
}

func reverse(nums []int, l int, r int) {
    for l < r {
        nums[l], nums[r] = nums[r], nums[l]
        l++
        r--
    }
}
```

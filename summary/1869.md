# 1869. Longer Contiguous Segments of Ones than Zeros

## [Description](https://leetcode.com/problems/longer-contiguous-segments-of-ones-than-zeros/)

## Solution

```go
// Ключевая идея:
// Считаем наибольший сегмент единиц и наибольший сегмент нулей, а затем
// сравниваем.
// Time Complexity: O(n) или o(2n)
// Space Complexity: O(1) или o(1)
func checkZeroOnes(s string) bool {
    return length(s, '1') > length(s, '0')
}

func length(s string, char byte) int {
    i, j, l := 0, 0, 0
    for j < len(s) {
        if s[i] != char {
            i++
            j++
            continue
        }
        for j < len(s) && s[j] == char {
            j++
        }
        l = max(l, j-i)
        i = j
    }
    return l
}
```

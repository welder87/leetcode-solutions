# 82. Remove Duplicates from Sorted List II

## [Description](https://leetcode.com/problems/remove-duplicates-from-sorted-list-ii)

## Solution

```go
func deleteDuplicates(head *ListNode) *ListNode {
    // строим новый список и не знаем заранее его голову, создаем Dummy Node
    // в случае, если в начале есть дубли это важно.
    dummy := &ListNode{Next: head}
    // два указателя на Dummy Node и исходный head
    prev, cur := dummy, head
    // условие цикла: пока указатель на текущий элемент не nil.
    for cur != nil {
        // условие второго цикла: пока указатель на следующий элемент у текущего
        // не nil. это предотвращает проверку в цикле выше и учитывает в коде
        // ниже
        for cur.Next != nil && cur.Val == cur.Next.Val {
            cur = cur.Next
        }
        // Если cur не сдвинулась, значит, дубликатов нет
        if prev.Next == cur {
            prev = cur
        } else {
            // Если cur сдвинулся, значит, есть дубликаты, и мы пропускаем их
            prev.Next = cur.Next
        }
        cur = cur.Next
    }
    // сохранив указатель на Dummy Node, просто возвращаем следующее, а значит
    // указатель на ответ.
    return dummy.Next
}
```

# 1984. Minimum Difference Between Highest and Lowest of K Scores

## [Description](https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/)

## Solution

```go
import (
     "sort"
     "math"
)
// Ключевая идея:
// Для сравнения чисел можно использовать brute force, но это затратно по
// памяти, поэтому лучше использовать сортировку, а затем Sliding Window с
// размером k.
// Time Complexity: O(n log n + n)
// Space Complexity:  O(log n), O(n), O(n log n)
// зависит от реализации сортировки в языке
func minimumDifference(nums []int, k int) int {
    // сортируем по возрастанию
    sort.Ints(nums)
    // создаем Sentinel
    ans := math.MaxInt32
    // создаем срез (массив не копируется) для отсекания последних k значений.
    for i, num := range nums[:len(nums)-k+1] {
        // находим минимальную разницу между наибольшим и наименьшим среди k
        // значений
        ans = min(ans, nums[i+k-1]-num)
    }
    return ans
}
```

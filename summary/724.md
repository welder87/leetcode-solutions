# 724. Find Pivot Index

## [Description](https://leetcode.com/problems/find-pivot-index/)

## Solution

```go

func pivotIndex(nums []int) int {
    sm := 0
    for i:= range len(nums) {
        sm += nums[i]
    }
    pxSum := 0
    for i := range len(nums) {
        if pxSum == sm - nums[i] - pxSum {
            return i
        }
        pxSum += nums[i]
    }
    return -1
}
```

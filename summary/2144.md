# 2144. Minimum Cost of Buying Candies With Discount

## [Description](https://leetcode.com/problems/minimum-cost-of-buying-candies-with-discount/)

## Solution

```go
import "sort"

// Ключевая идея:
// Для сравнения чисел можно использовать brute force, но это затратно по
// памяти, поэтому лучше использовать сортировку, а затем Sliding Window с
// размером 3.
// Time Complexity: O(n log n + n)
// Space Complexity:  O(log n), O(n), O(n log n)
// зависит от реализации сортировки в языке

func minimumCost(cost []int) int {
    // сортируем по убыванию
    sort.Slice(cost, func(i, j int) bool {
        return cost[i] > cost[j]
    })
    ans := 0
    // движемся по массиву по 3 значения
    for i := range len(cost) {
        // в случае если остаток от деления на 3 не равен 2 - суммируем
        // [9,7,6,5,2,2]
        // 0 % 3 = 0
        // 1 % 3 = 1
        // 2 % 3 = 2
        // 2 % 3 = 0
        if i%3 != 2 {
            ans += cost[i]
        }
    }
    return ans
}
```

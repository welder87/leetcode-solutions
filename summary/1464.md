# 1464. Maximum Product of Two Elements in an Array

## [Description](https://leetcode.com/problems/maximum-product-of-two-elements-in-an-array/)

## Solution

```go
// Ключевая идея:
// Находим первый и второй максимумы двигаясь вдоль массива.
func maxProduct(nums []int) int {
    // инициализируем Sentinel Value, по условиям задачи достаточно ноля.
    mx1, mx2 := 0, 0
    for _, num := range nums {
        if mx1 < num {
            mx2 = mx1
            mx1 = num
        } else if mx2 < num {
            mx2 = num
        }
    }
    return (mx1 - 1) * (mx2 - 1)
}
```

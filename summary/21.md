# 21. Merge Two Sorted Lists

## [Description](https://leetcode.com/problems/merge-two-sorted-lists/)

## Solution

```go
// Стандартное итеративное решение, которое можно реализовать в любом другом языке.
func mergeTwoLists(list1 *ListNode, list2 *ListNode) *ListNode {
    // строим новый список и не знаем заранее его голову, создаем Dummy Node
    dummy := &ListNode{}
    // задаем текущий элемент списка
    cur := dummy
    // условие цикла: пока оба из указателя на списки не nil.
    for list1 != nil && list2 != nil {
        // сравниваем значения.
        // не важно будет <= или просто < , суть не меняется.
        if list1.Val <= list2.Val {
            cur.Next = list1
            list1 = list1.Next
        } else {
            cur.Next = list2
            list2 = list2.Next
        }
        cur = cur.Next
    }
    // по выходу из цикла у нас или оба указателя nil или один из низ не nil.
    // и в итоге просто присваиваем конец ненулевого списка, не проходя по нему.
    if list1 != nil {
        cur.Next = list1
    } else {
        cur.Next = list2
    }
    // сохранив указатель на Dummy Node, просто возвращаем следующее, а значит
    // указатель на ответ.
    return dummy.Next
}
```
